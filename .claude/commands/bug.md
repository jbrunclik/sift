Investigate and fix a bug: $ARGUMENTS

Steps:
1. Reproduce or understand the issue from the description.
2. Search the codebase for the relevant code paths.
3. Identify the root cause.
4. Implement the fix — keep it minimal, don't refactor unrelated code.
5. Add or update a test that covers the bug.
6. Run `make lint`, `make typecheck`, `make test`. Fix any issues.
7. Summarize: what was the root cause, what was changed, how to verify.
